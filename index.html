nativetable.js - demo

nativetable is a little library ready to help you to easily display your data in HTML table.

The samples below show how to use nativetable to filter, sort and paginate data.

Pagination

Sample code for pagination 

          let nt = new Nativetable('nativetable', {
            sources: data,
            pagination: {
              maxLength: 5
            }
          })

Filtering

Sample code for filtering data 

          let nt = new Nativetable('nativetable', {
            sources: data,
            pagination: {
              maxLength: 5
            },
            filters: {
              $and: {
                name: ['julie', 'sarah'],
                age: (age) => {
                  return age >= 29 && age <= 39
                }
              },
              $or: {
                man: [false],
                brothers: [1]
              }
            }
          })

Sorting

Sample code for sortable table 

          let nt = new Nativetable('nativetable', {
            sources: data,
            sorting: true
          })
